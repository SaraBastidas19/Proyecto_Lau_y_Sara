## 🔄 Data Flow

```
                    BROWSER
                      │
                      ▼
        ┌─────────────────────────┐
        │    REACT FRONTEND      │
        │  (http://localhost:3000)│
        │                         │
        │  ┌──────────────────┐   │
        │  │  Page Components │   │
        │  │  (Home, Services)│   │
        │  └────────┬─────────┘   │
        │           │             │
        │  ┌────────▼─────────┐   │
        │  │   Axios Client   │   │
        │  │   (api.js)       │   │
        │  └────────┬─────────┘   │
        └───────────┼─────────────┘
                    │
                    │ HTTP Request
                    │ (POST /api/contact)
                    │
                    ▼
        ┌─────────────────────────┐
        │   EXPRESS BACKEND      │
        │ (http://localhost:5000) │
        │                         │
        │  ┌──────────────────┐   │
        │  │  Routes (Router) │   │
        │  │  /api/contact    │   │
        │  └────────┬─────────┘   │
        │           │             │
        │  ┌────────▼─────────┐   │
        │  │  Controllers     │   │
        │  │  submitContact() │   │
        │  └────────┬─────────┘   │
        │           │             │
        │  ┌────────▼─────────┐   │
        │  │  Validation      │   │
        │  │  Error Handling  │   │
        │  └────────┬─────────┘   │
        │           │             │
        │  ┌────────▼─────────┐   │
        │  │  File System     │   │
        │  │  messages.json   │   │
        │  └──────────────────┘   │
        └─────────────────────────┘
                    │
                    │ JSON Response
                    ▼
                 BROWSER
```
